# Personal Expense Tracker (Python)

## Description
A Python console application to track daily expenses, categorise spending, and analyse monthly totals. Built as a placement-ready project to demonstrate problem-solving and Python skills.

## Features
- Add expenses with amount, category, and note
- View all recorded expenses
- Calculate total spending
- Filter expenses by category
- Show monthly summary
- Persistent storage using CSV files

## Technologies
- Python (functions, lists, dictionaries, file handling)
- CSV file handling for data persistence

## Learning Outcomes
- Applied Python programming concepts
- Developed logical thinking and modular code
- Gained experience in real-world problem solving

# Library Management System (Python / DSA)

## Description
A console-based Library Management System to demonstrate core Data Structures & Algorithms using Python. Handles book addition, removal, issuing, and searching using lists, stack, and queue operations.

## Features
- Add and display books
- Remove last added book (Stack)
- Issue books to students (Queue)
- Search books by name
- Easy-to-understand console interface

## Data Structures Used
- List → store books
- Stack → removed books
- Queue → issued books

## Learning Outcomes
- Applied DSA concepts in a real-world scenario
- Practiced Python functions and modular code
- Improved problem-solving and logical thinking
